# 🍕 Pizza Sales Analysis Using SQL

## 📌 Project Overview

This project focuses on analyzing pizza sales data using SQL to extract meaningful insights and solve real-world business problems.

The analysis includes revenue trends, order patterns, and product performance to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing pizzas and categories
* Understand customer order behavior
* Evaluate revenue trends over time

---

## 🛠️ Tools Used

* SQL (Structured Query Language)
* Database Management System (MySQL / SQL Server)

---

## 📂 Dataset Description

The dataset contains information about:

* Orders
* Order details
* Pizza types
* Categories
* Prices and quantities

---

## 🔍 SQL Analysis

### ✅ Basic Level

* Total number of orders
* Total revenue generated
* Highest priced pizza

### ✅ Intermediate Level

* Top 5 most ordered pizzas
* Most popular categories
* Order distribution by time

### ✅ Advanced Level

* Revenue contribution by pizza type
* Cumulative revenue over time
* Top pizzas by category

---

## 📊 Key Insights

* Peak order time occurs during evening hours, indicating high demand
* A small number of pizzas generate a large portion of revenue
* Certain categories perform better in both quantity and sales
* Order patterns remain consistent with slight variations

👉 These insights help in business decision-making and improving sales strategies.

---

## 📌 Conclusion

This project demonstrates how SQL can be used to analyze business data and extract actionable insights.
